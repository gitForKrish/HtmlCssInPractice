3. Bold Text

Strong tag is used to indicate that the text is fundamentally or semantically important to the surrounding text. A different tone of voice text-to-speech program uses a different tone of voice to convey that the text is important.

4. Italic Text

em tag is used to indicate that the text should have extra emphasis that should be stressed, while i tag simply represents text which should be set off from the normal text around it.

5. Underlined Text

u tag in HTML 5 is used to represent an unarticulated, non-textual annotation. You might use such a rendering to indicate mispelled text on the page, or for a Chinese proper name mark.
